## Installation

YLBModule is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'YLBModule'
```

获取最新代码：
```ruby
pod 'YLBModule', :git => 'https://github.com/ProBobo/YLBModule.git'
```
